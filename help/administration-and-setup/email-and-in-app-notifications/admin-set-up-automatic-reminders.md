# 자동 미리 알림 설정

자동 미리 알림은 모든 오브젝트에 적용되고, 해당 작업 항목에 할당된 모든 사람에게 전송되며, 메시지를 사용자 정의할 수 없다는 점을 제외하면 미리 알림 과 유사합니다.

다음 중 하나 이상으로 보낼 수 있습니다.

* 작업 또는 문제에 할당된 사용자입니다.
* 사용자의 직접 관리자.
* 직속 관리자의 관리자입니다.

일반적으로 사용되지는 않지만 조직에서 문제가 있고 사용자가 정시에 작업을 완료했다는 책임을 묻고자 하는 경우 자동 미리 알림이 유용할 수 있습니다.

자동 미리 알림은에서 설정할 수 있습니다. [!UICONTROL 이메일] 의 섹션 [!UICONTROL 설정] 영역입니다. 다음을 결정하기만 하면 됩니다.

* 자동 알림 필요
* 작업 기한 전/후 시간이 얼마나 됩니까? 알림이 전송되어야 합니다.

![[!UICONTROL 자동 미리 알림] 창 위치 [!UICONTROL 설정]](assets/admin-fund-automatic-reminders-1.png)

이 시점에서 자동 미리 알림이 적용되었으며 기준이 충족되는 즉시 전송됩니다. 또한 사용자는 자신의 계정에서 자동 미리 알림을 비활성화할 수 없습니다 [!UICONTROL 내 설정] 창.
