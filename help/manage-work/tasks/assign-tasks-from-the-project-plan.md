# 프로젝트 계획에서 작업 할당

이 비디오를 통해 다음과 같은 사항을 알아볼 수 있습니다.

* 인라인 편집을 사용하여 작업을 할당하는 방법을 알아봅니다
* 동일한 사람에게 여러 작업을 할당하는 방법을 알아봅니다
* 작업 할당 시 작업 역할의 가치 이해
* 작업 할당 시 프로젝트 템플릿의 가치 이해
* 팀에 작업을 할당함으로써 얻을 수 있는 가치 이해
