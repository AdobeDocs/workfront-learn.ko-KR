# 기존 프로젝트를 효율적으로 복사

이 비디오에서는 템플릿을 사용하는 대신 프로젝트를 복사하는 방법을 설명합니다. 프로젝트&#x200B;을 복사하려면 원하는 프로젝트를 열고 프로젝트 이름 옆에 있는 세 점 메뉴를 클릭한 다음 &quot;복사&quot;를 선택합니다. 프로젝트 복사 창&#x200B;에서 프로젝트 이름을 바꾸고 상태를 변경할 수 있습니다. 지정된 사용자에게 통지를 발송하지 않도록 하려면 일반적으로 &quot;계획&quot;으로 변경합니다. 할당, 진행 상황, 사용자 지정 데이터 등을 복사할지 여부를 선택할 &#x200B; 수 있습니다&#x200B;.
완료하려면 &quot;프로젝트 복사&quot;를 클릭합니다. &#x200B;
문제는 계획된 작업으로 간주되지 않으므로 이 프로세스 중에 직접 복사할 수 없습니다. 문제&#x200B;을 복사하려면 문제 탭으로 이동하여 원하는 문제를 선택한 다음, &quot;복사 위치&quot; 옵션을 사용하여 해당 문제를 새 프로젝트로 전송합니다. &#x200B;


>[!VIDEO](https://video.tv.adobe.com/v/3456045/?captions=kor&quality=12&learn=on&enablevpops=1)

## 주요 학습 사항

* **프로젝트 복사:** 프로젝트를 열고 프로젝트 이름 옆에 있는 세 점 메뉴를 클릭한 다음 &quot;복사&quot;를 선택하여 복제합니다. &#x200B;
* **프로젝트 설정 조정 중:** 할당된 사용자에게 알림을 보내지 않도록 프로젝트 이름을 변경하고 상태를 &quot;계획 중&quot;으로 변경하십시오.
* **선택적 복사:** 할당, 진행 상황 또는 사용자 지정 데이터를 복사할지 여부를 선택할 수 있습니다. 이 &#x200B; 옵션을 선택 해제하면 새 프로젝트에서 사용자 지정할 수 있습니다.
* **사용자 지정 데이터 처리:** 사용자 지정 데이터를 선택 취소하면 양식이 데이터 없이 복사됩니다. 양식과 해당 데이터를 모두 복사하려면 상자를 &#x200B; 선택한 상태로 둡니다. &#x200B;
* **문제를 별도로 복사:** 문제는 프로젝트 복제 프로세스 중에 복사되지 않습니다. 복사하려면 &#x200B; 문제 탭으로 이동하여 원하는 문제를 선택한 다음, &quot;복사 위치&quot; 옵션을 사용하여 새 프로젝트로 전송합니다. &#x200B;
