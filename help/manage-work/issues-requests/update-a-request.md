# 요청 업데이트

이 비디오에서는 다음 방법을 배우게 됩니다.

* 제출한 요청 찾기
* 요약 패널에서 업데이트 수행하기
* 요청 페이지 열기
* [!UICONTROL 업데이트] 패널에서 요청 업데이트하기
