# 문제 할당 관리

문제 관리는 프로젝트 관리자와 해당 문제에 대해 작업하도록 할당된 사람들의 중요한 책임입니다.

이 비디오에서는 다음 방법을 배우게 됩니다.

* 문제 할당
* 문제 할당 찾기
* 문제에 대한 진행 상황 업데이트
* 문제 상태 업데이트
