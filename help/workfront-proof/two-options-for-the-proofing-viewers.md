# 교정쇄 뷰어 설명

[!DNL Workfront]를 사용하여 정적 및 인터랙티브한 두 가지 주요 교정쇄 카테고리를 검토할 수 있습니다.

정적 교정쇄는 PDF, 이미지 및 텍스트 파일과 같은 것입니다. 일부 웹 사이트 교정쇄는 교정쇄가 업로드된 방법에 따라 인터랙티브 교정쇄 카테고리에 속합니다.

[!DNL Workfront]는 보고 있는 교정쇄 유형(및 교정 시스템 관리자가 수행한 일부 설정)에 따라 웹 기반 교정쇄 뷰어 또는 데스크탑 교정쇄 뷰어의 해당 버전을 엽니다.

웹 기반 교정쇄 뷰어는 정적 교정쇄 및 비디오의 [!DNL Workfront’s] 기본값입니다. 이메일 또는 [!DNL Workfront] 자체에서 [!UICONTROL 교정쇄 열기]를 클릭하면 [!DNL Workfront] 브라우저 창의 일부로 웹 기반 뷰어가 열립니다.

데스크탑 교정쇄 뷰어는 워크스테이션에서 실행되는 독립 실행형 애플리케이션입니다. 주로 URL, 임베디드 또는 스트리밍 비디오, 인터랙티브 배너 및 HTML5 애니메이션을 사용하여 만든 인터랙티브 웹 사이트 교정쇄에 사용됩니다. 뷰어를 사용하면 웹 사이트의 일부인 링크와 메뉴를 클릭하고 다른 페이지로 이동할 수 있습니다.

두 개의 뷰어는 동일하게 작동하므로 교정쇄 유형에 관계없이 검토 시 공동 작업이 보장됩니다.

## 데스크탑 교정쇄 뷰어 설치

인터랙티브 콘텐츠가 포함된 교정쇄를 처음 열면 데스크탑 교정쇄 뷰어를 다운로드하라는 메시지가 표시됩니다. 지침을 따르면 됩니다.

조직에서 워크스테이션에 애플리케이션 설치를 허용하지 않는 경우, 시스템 관리자와 협력하여 필요한 교정쇄 뷰어가 있는지 확인합니다.

데스크탑 교정쇄 뷰어 애플리케이션이 설치되면 나중에 인터랙티브 교정쇄가 열립니다. 한 번만 다운로드하면 됩니다.
