# 전역 교정쇄 설정 지정

[!DNL Workfront]에서는 사용자에게 영향을 미치는 전역 기본 설정을 지정하여 교정쇄를 위한 시스템 설정을 시작합니다. 이러한 기본값을 설정하면 만들어진 사용자가 교정 기능을 사용하기 시작하는 데 필요한 최소 설정이 확보되며, 교정쇄 설정을 통해 작업할 때 시간을 절약할 수 있습니다.

* 기본 교정 역할 할당
* 기본 계정 설정 지정
* 기본 교정쇄 설정 지정

이러한 기본값을 설정하기 전에 이미 사용자 만들기를 시작했더라도 걱정하지 마십시오. 개별 사용자 프로필에서 이 모든 정보를 편집할 수 있습니다.
