# 스위치 함수 연습

간단한 데이터 변경의 경우 Switch 함수를 사용하여 모듈 필드 내에서 한 값을 다른 값으로 변환합니다. 이 연습에서는 두 글자로 된 키를 전자 메일로 보낸 프로젝트 진행 상태의 실제 이름으로 변경합니다.
