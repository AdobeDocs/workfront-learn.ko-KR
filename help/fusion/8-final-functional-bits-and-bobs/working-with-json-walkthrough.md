# JSON 작업 연습

## 개요

![Fusion 시나리오의 이미지](assets/final-functional-bits-and-bobs-2.png)

## 배열 연습

이 비디오에서는 다음 방법을 배웁니다.

* 시나리오 내에서 JSON을 만들어 구문 분석하여 디자인 요구 사항을 지원합니다
